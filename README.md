## Qëllimi i projektit
Qëllimi i këtij projekti është krijimi i një grafiku duke përdorur gjuhën e programimit Python dhe libraritë për analizë dhe vizualizim të të dhënave.

## Udhëzime për ekzekutimin e kodit

1. Shkarkoni repository nga GitHub.

2. Instaloni libraritë e nevojshme me komandën:

pip install -r requirements.txt

3. Ekzekutoni programin nga terminali me komandën:

python src/main.py

## Përshkrimi i figurës

Figura e krijuar paraqet grafikun e funksionit matematikor sin(x). 
Grafiku gjenerohet duke përdorur librarinë matplotlib dhe ruhet në folderin figures me emrin signature_plot.png.
